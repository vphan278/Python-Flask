# Python Flask Practice Projects

This repository contains practice projects built while learning the Flask web framework in Python.

The projects focus on understanding how Flask works, including routing, templates, sessions, and handling user requests.

--------------------------------------------------

PROJECTS INCLUDED

Hello Flask
A basic Flask application demonstrating routing and server responses.

Understanding Routing
Examples showing how URL routes are handled in Flask applications.

HTML Rendering
Examples demonstrating how Flask renders HTML templates.

Post
Examples showing how to handle form submissions using POST requests.

Redirecting
Examples demonstrating page redirection in Flask.

Checkerboard
A simple application demonstrating dynamic page generation using Flask templates.

Counter
A project demonstrating how sessions can track page visits.

Session Demo
Examples demonstrating how session variables work in Flask.

Dojo Fruit
A small application demonstrating form handling and dynamic content.

Dojo Fruit Store
A simple store-style application demonstrating user input and calculations.

Recipe
A basic web application demonstrating user input and template rendering.

Pies
A simple project demonstrating Flask routing and HTML templates.

Scuba Diving
A small application demonstrating page rendering and routing.

Playground
Examples showing how Flask can dynamically generate pages.

FEATURES PRACTICED

GOAL
